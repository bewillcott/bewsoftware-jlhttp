MDj CLI

This is a command-line program that uses the MDj Core library.

It will process either a single file, all the files in a directory, or all the files in a directory tree.

The default extensions are:

  • Markdown input files: .md
  • HTML output files: .html

The program has a number of options. For details see the Options page.
